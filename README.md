# install
```
bash <(curl -Ls https://raw.githubusercontent.com/M946i/s-ui/refs/heads/main/install-s-ui.sh)
```
# reset username password
```
/usr/local/s-ui/sui admin -username 111 -password 111
```
# setting port path subPort subPath
```
/usr/local/s-ui/sui setting -port 2095
/usr/local/s-ui/sui setting -path /app/
/usr/local/s-ui/sui setting -subPort 2096
/usr/local/s-ui/sui setting -subPath /sub/
```
